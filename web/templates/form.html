FAQ

Что показывает проект

Коротко о том, какие backend-механики здесь реализованы.

Frontend отправляет JSON через fetch, backend валидирует данные, сохраняет заявку и возвращает результат клиенту.

После входа backend проверяет session cookie, восстанавливает пользователя и разрешает обновлять только собственную заявку.

Админ может смотреть заявки, редактировать их, удалять и анализировать статистику по выбранным языкам программирования.

В нем есть не только CRUD, но и authentication flow, middleware, sessions, MySQL repository, relational loading и frontend integration.
